#### Record ubuntu terminal

```
# install asciinema

$ sudo apt install asciinema -y

# download agg

# https://github.com/asciinema/agg/releases

# record

$ asciinema rec #file name
$ asciinema play #file name
$ agg #file name #gif filename
```
